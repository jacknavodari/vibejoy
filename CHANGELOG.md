# Changelog

All notable changes to the VibeJoy Media Player project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-07-24

### Added
- Initial release of VibeJoy Media Player
- Modern dark theme GUI with enhanced visual design
- Album art display functionality
- Track information panel showing metadata
- Progress bar with time display
- Playlist management features
- Playback controls (play, pause, stop, next, previous)
- Shuffle and repeat modes
- Volume control slider
- Folder import functionality
- Standalone Windows executable build
- Support for MP3, WAV, and OGG audio formats
- JSON-based playlist persistence

### Changed
- Completely redesigned user interface with improved layout
- Enhanced visual styling and theming
- Better organization of controls and information display

### Fixed
- Various UI/UX improvements for better user experience
- Improved error handling for audio file operations
